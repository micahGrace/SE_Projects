This is a Dictionary application. This Application enables
the user to Add a new word to the dictionary 
in addition to providing the common services a dictionary can provide.
you can search for a particular word by its word or its description.

=> Langueage - Java
=> Architecture - Mvc Architecture

This Application has:
    - Model - Has a model class  which stores
              the words and description in MICROSOF_ACCESS database.
    - Controller - Java 
    - View - Java Swing

Everything is done as the Instructor Commands...
